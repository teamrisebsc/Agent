## How to Help

**Pipeline overview:**
1. Pull all recruits currently in pre-license (have PreLicense date but no Completed date)
2. Group by stage: enrolled, test scheduled, waiting on results, passed, needs reschedule
3. Flag anyone whose test date is coming up within 7 days or is overdue

**Auto messages — draft templates for:**
- Pre-license enrollment confirmation
- Weekly study check-in
- Test date reminder (3 days before)
- Post-test encouragement (pass or fail)
- License activation next steps

Always keep messaging warm, motivating, and aligned with WFG culture — getting licensed is a life-changing milestone.
