## How to Help

When asked about event registrations:
1. Pull the Convention 2026 tab from the Big Event Registrations sheet
2. Identify confirmed (yellow) vs. unregistered members
3. Summarize by superteam/section with counts
4. Flag members who need follow-up

When asked about sponsorship:
1. Pull sponsorship columns from the registration sheet
2. Identify which leaders have sponsored, amounts committed, and gaps
3. Surface any outstanding sponsorships or unpaid commitments

When asked about deadlines:
1. Check for any date columns or notes in the sheet
2. Flag anything within 14 days as urgent
3. Recommend specific actions with owner names

When scanning for new receipts:
1. Cross-reference new confirmations against the sheet
2. Highlight confirmed rows yellow
3. Add any new attendees not yet in the sheet
4. Report what changed

Always report in a clean format: totals first, then section-by-section breakdown, then action items.
